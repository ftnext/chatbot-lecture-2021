環境構築¶

現在のドキュメントのバージョンは、macOSを前提としています。

Pythonのバージョン¶

確認しましょう

$ python3 -V

（構築した方法によっては python かもしれません）

Python 3.7.x で進めます。

インストールしていない場合は Python 3.7 インストール方法

作業ディレクトリを作る¶

以下のコマンドのディレクトリのパスは一例です。

$ mkdir -p ~/programming/chatbot
$ cd ~/programming/chatbot

仮想環境¶

作業ディレクトリにいる状態を前提にしています（pwd で確認しましょう）。

$ python3.7 -m venv env
$ source env/bin/activate
(env) $ python -m pip install -U pip

プロンプトの (env) は仮想環境が有効になった状態を示します。

Tip

source コマンドは . と簡略化できます（. env/bin/activate）。

env/bin/activate はシェルスクリプトで、source コマンドで実行しています。

注釈

仮想環境を無効にする場合は deactivate コマンドを入力します。

ChatterBot インストール¶

(env) $ pip install ChatterBot

2021/11時点では 1.0.8 がインストールされます。

この時点では対話モードで import chatterbot してもエラー（ModuleNotFoundError）が送出されます。 次の項目にて、依存するライブラリをインストールすることで解決します。
